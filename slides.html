🛠 Под капотом

Чем агент отличается от чат-бота

Обычный промптинг vs Агент

Чат-ботАгент
КодПишет, но не запускаетПишет и выполняет
Данные«Расскажи про данные»Загружает, парсит, считает
ИтерацияОдин вопрос — один ответЦепочка шагов, уточнения
ОшибкиГаллюцинируетВидит ошибку → чинит → повторяет
РезультатТекстЦифры, графики, код

Как работает агент

Контекст → задаёшь роль, описываешь продукт

Инструменты → Python, файлы, браузер, API

Цикл → думает → пишет код → выполняет → анализирует → повторяет

По сути, junior-аналитик, которому не нужно спать

Секрет успеха: контекст

«Ты — аналитик FitFlow. Фитнес по подписке, Россия. events.csv — 107K. Найди причины оттока.»

3 принципа

1
Контекст решает
Чем лучше описал продукт — тем точнее анализ
2
Итерация, не одноразовый запрос
Хороший анализ — диалог из 5-15 сообщений
3
Доверяй, но проверяй
Агент может ошибиться — проси показать данные
