# Агентная аналитика для продактов

Материалы мастер-класса по агентной аналитике — как AI-агент формулирует гипотезы, проверяет их кодом, анализирует фидбек пользователей, всё в одном диалоге.

## Сайт

**[working-in-it.github.io/agentic-analytics-workshop](https://working-in-it.github.io/agentic-analytics-workshop)**

## Что внутри

- **Слайды** — интерактивная презентация (reveal.js)
- **Промпты для заданий** — готовые промпты с кнопкой Копировать
- **Датасет FitFlow** — синтетические данные фитнес-приложения

## Инструменты

- Google AI Studio — бесплатно, без VPN
- Cursor — AI-редактор, бесплатный тир
- Claude.ai — нужен VPN

## Три принципа

1. **Контекст решает**
2. **Итерация, не одноразовый запрос**
3. **Доверяй, но проверяй**
