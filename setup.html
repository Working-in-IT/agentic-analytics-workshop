Установка Qwen Code

Настройте бесплатный AI-агент в VS Code за 5 минут

⚠ Пожалуйста, сделайте это до мастер-класса — на площадке времени на установку не будет

1

Установить VS Code

Скачайте установщик с code.visualstudio.com.

Откройте скачанный .dmg-файл и перетащите иконку VS Code в папку Applications.

2

Установить расширение Qwen Code

Откройте VS Code, затем панель расширений: Cmd + Shift + X.

В строке поиска введите Qwen Code и нажмите Install.

Или установите напрямую из браузера:
marketplace.visualstudio.com → Qwen Code (Tongyi Lingma)
3

Войти в аккаунт

Если у вас ещё нет аккаунта — зарегистрируйтесь на chat.qwen.ai.

В VS Code кликните на иконку Qwen Code в боковой панели и нажмите Sign In. Откроется браузер — авторизуйтесь через свой аккаунт Qwen Chat.

Важно: после авторизации в браузере вернитесь в VS Code — он подхватит сессию автоматически.
4

Скачать папку воркшопа

Скачайте fitflow-workshop.zip и распакуйте в удобное место.

Откройте эту папку в VS Code: File → Open Folder → выберите распакованную папку.

В папке уже есть данные для анализа и файл CLAUDE.md с контекстом — Qwen Code подхватит его автоматически.
5

Проверить, что всё работает

Откройте чат Qwen Code в боковой панели и напишите:

Привет, напиши print('hello')

Если агент ответил и предложил код — установка прошла успешно.

Проблемы?

Попробуйте искать по альтернативным названиям: Qwen Code или Tongyi Lingma. Убедитесь, что у вас актуальная версия VS Code. Если не помогло — установите расширение напрямую через браузер: ссылка на маркетплейс.
Убедитесь, что у вас есть аккаунт на chat.qwen.ai. Если браузер открылся, но авторизация не прошла — попробуйте войти в Qwen Chat напрямую в браузере, а затем повторите Sign In из VS Code.
Проверьте интернет-соединение. Попробуйте перезапустить VS Code. Если проблема сохраняется — выйдите из аккаунта и войдите заново через Sign In.
